- Comment démarer le site web ?

--> Afin de démarer le site web, il vous suffit simplement de récupérer le dossier complet "Projet_Darkmode".
Ensuite faite un clic droit sur le fichier index.html > ouvrir avec.. et séléctionnez le navigateur de votre choix ou simplement double-cliquez sur le fichier. 

- Comment fonctionne le site web ?

--> Le fonctionnement du site est relativement simple. Nous vous invitons simplement à suivre les instructions données sur le site.
